Amortized Roofline Analyzer

By Mona Mishra · Extension of the Roofline Model for MoE & Speculative Decoding Workloads

The key insight from recent research: The standard Roofline Model assumes dense, sequential token generation. Modern workloads break this assumption in two important ways — (1) Mixture-of-Experts (MoE) activates only a fraction of parameters per token, reshaping the memory access pattern; (2) Speculative Decoding amortizes the cost of the large verifier model across multiple verified tokens, raising effective arithmetic intensity. This tool makes both effects visible on the Roofline chart.

Based on: MoE-SpeQ (arXiv 2511.14102) · SPAD (arXiv 2510.08544) · LLM Inference Limits (arXiv 2507)

Speculative Decoding — Amortized Roofline
Speculative decoding uses a small draft model to propose γ tokens, then a large verifier validates them in parallel. The verifier's forward pass over γ tokens is matrix-matrix (not matrix-vector) — increasing arithmetic intensity. This "amortization" moves the operating point rightward on the Roofline.
